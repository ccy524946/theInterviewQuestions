### 1.5 . int 跟 Integer有什么区别？

| Integer                                                      | int                    |
| ------------------------------------------------------------ | ---------------------- |
| int的包装类                                                  | java的一种基本数据类型 |
| 必须实例化后才能使用                                         | int不需要              |
| 实际是对象的引用，当new一个Integer时，实际上时生成一个指针指向此对象 | 直接存储数据值         |
| 默认值是null                                                 | 默认值是0              |
